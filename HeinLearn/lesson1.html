PHP ဆိုတာဘာလဲ?

PHP သည် server side scripting language တစ်ခုဖြစ်သည်။ scripting language သည် run time တွင်လုပ်ဆောင်သော program တစ်ခုဖြစ်သည်။ ဆိုလိုသည်မှာ ဤပရိုဂရမ်များကို C/C++ သို့မဟုတ် Java တို့နှင့်မတူဘဲ စုစည်းမှုမလိုအပ်ပါ။ ဆာဗာဘက်တွင်၊ JavaScript ကဲ့သို့ client side (browser) တွင် တိုက်ရိုက်မလည်ပတ်နိုင်ဟု ဆိုလိုပြီး ၎င်းကို လုပ်ဆောင်သည့် ဆာဗာတစ်ခု လိုအပ်သည်။

PHP Architecture

PHP ကဘာတွေလုပ်လဲ?

PHP ကို ​​လုပ်ဆောင်ချက် နှစ်ခုအတွက် အဓိက အသုံးပြုသည် ။
1. ဝဘ်ဆိုဒ် ဖွံ့ဖြိုးတိုးတက်ရေး- Zend Engine ကိုအသုံးပြု၍ PHP ကုဒ်ကို HTML နှင့် CSS သို့ပြောင်းသည်။
2. SQL နှင့် back end processing - scripts များအတွင်းရှိလုပ်ဆောင်ချက်များကို backend processing အတွက်အသုံးပြုပြီး SQL ကို server မှခွဲခြမ်းစိတ်ဖြာသည်။

အထက်ဖော်ပြပါ လုပ်ဆောင်ချက်များအပြင် PHP ကို ​​သီးခြားဂရပ်ဖစ်အက်ပ်လီကေးရှင်းများ၊ Robotic Drone ထိန်းချုပ်မှုများအဖြစ်လည်း အသုံးပြုနိုင်သည်။ WordPress၊ Magento၊ Drupal နှင့် Joomla ကဲ့သို့သော အကျော်ကြားဆုံး အကြောင်းအရာ စီမံခန့်ခွဲမှုစနစ်များသည် PHP ပေါ်တွင် လုပ်ဆောင်သည်။

PHP Programကိုဘယ်လို run ရမလဲ?

HTML အကြောင်းအရာအတွင်းမှ php ပရိုဂရမ်တစ်ခု သို့မဟုတ် php ပရိုဂရမ်များသာရှိသော php ပရိုဂရမ်တစ်ခုကို လုပ်ဆောင်ရန်၊ ၎င်းတို့ကို .php ဖိုင်များအဖြစ် သိမ်းဆည်းပါသည်။ php ဖိုင်ကို server side executable file တစ်ခုဖြစ်သောကြောင့် ကျွန်ုပ်တို့သည် ဆာဗာတစ်ခု လိုအပ်ပါသည်။ ကျွန်ုပ်တို့သည် ဤနေရာတွင် ဖော်ပြထားသော အဆင့်များကို ဦးစွာလိုက်နာပါမည်-
"XAMPP ဖြင့် PHP တပ်ဆင်ခြင်း"။ XAMPP ကို ​​တပ်ဆင်ပြီးသည်နှင့် ဆာဗာလည်ပတ်နေသည်နှင့် တပြိုင်နက်၊ ကျွန်ုပ်တို့သည် windows ရှိ “C:\xampp\htdocs” အတွင်းရှိ “test” ဖိုင်တွဲတစ်ခုတွင် ကျွန်ုပ်တို့၏ php ဖိုင်များကို သိမ်းဆည်းပါမည်။
ပြီးရင်။http://localhost:/test/ browser ထဲမှာ။ အထက်ဖော်ပြပါ ဆွေးနွေးထားသော php ကုဒ်ကို index.php တွင် သိမ်းဆည်းထားပါက၊ ၎င်းသည် ဤကဲ့သို့ လုပ်ဆောင်နိုင်လိမ့်မည်-
